# 个人图床 - lipot0509-lang/images

这个仓库用于存储我个人博客/微信公众号文章中的图片，通过 jsDelivr CDN 加速访问。

## 🌐 CDN 加速访问链接格式

所有图片可通过以下格式访问：https://cdn.jsdelivr.net/gh/lipot0509-lang/images@main/文件名


### 示例
- 图片文件：`20260227100040795.jpg`
- 访问链接：`https://cdn.jsdelivr.net/gh/lipot0509-lang/images@main/20260227100040795.jpg`

## 📁 文件命名规范

为避免编码问题，请遵循以下命名规则：

| 规则 | 说明 | 示例 |
|------|------|------|
| ✅ **使用英文/数字** | 避免中文、特殊字符 | `chengdu-2026.jpg` |
| ✅ **小写字母** | 统一使用小写 | `beijing-trip.png` |
| ✅ **连字符分隔** | 用 `-` 代替空格 | `sunset-at-sea.jpg` |
| ❌ **避免中文** | 可能导致上传失败 | ~~屏幕截图.jpg~~ |
| ❌ **避免空格** | 空格会被编码为 `%20` | ~~my photo.jpg~~ |

推荐格式：`地点-日期-描述.jpg`（如 `chengdu-2026-02-27-cloud.jpg`）

## 🚀 上传工具

本仓库使用 **PicGo** 配合 GitHub 图床进行图片管理：
- 仓库名：`lipot0509-lang/images`
- 分支名：`main`
- 自定义域名：`https://cdn.jsdelivr.net/gh/lipot0509-lang/images@main`
